About Groupie Tracker Project

Description: This is a web application made using Golang, HTML and CSS that allows a user to get artist info when clicked upon it's icon/name.

Authors: Karim Mouawad and Husain Alabbad

Usage: How to Run

  1. 1. Git clone the groupie-tracker repository
  2. 2. In your terminal, cd into the "main" folder
  3. 3. Run "go run ." in the terminal
  4. 4. Now open a browser and enter "localhost:8080" into the URL Address Bar
  5. 5. Click on the artists' icon or name
  6. 6. You will be redirected to a page with all the relevant artist info
